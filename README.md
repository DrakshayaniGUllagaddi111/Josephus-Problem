# 🔄 Josephus Problem (C Program)

This project implements the classic **Josephus Problem** using the C programming language. It determines the safe position in a circle of people where every k-th person is eliminated until only one survives.

---

## 🚀 Features

- Solves the Josephus Problem efficiently
- Takes input for number of people (n) and step count (k)
- Displays the safe position
- Simple and easy-to-understand logic

---

## 🛠️ Tech Stack

- C Programming

---

## ⚙️ How It Works

- n people stand in a circle
- Every k-th person is eliminated
- The process continues until only one person remains
- The program calculates the position of the last surviving person

## ▶️ How to Run

1. Open terminal and navigate to project folder:

cd josephus-problem


2. Compile the program:

gcc josephus.c -o josephus


3. Run the program:

./josephus


4. Enter values for n (people) and k (step count)

---

## 🎯 Example

Input:

n = 5, k = 2


Output:

Safe position: 3
